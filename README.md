## 📂 About This Portfolio Site

This repo contains the source code for my personal portfolio website.

### Tech Stack
- **React 19** + **Vite** — fast dev experience, optimized builds
- **Tailwind CSS 4** — utility-first styling with custom design tokens
- **Framer Motion** — subtle entrance animations, scroll-triggered reveals
- **Lucide React** — clean, consistent iconography

### Design Philosophy
- **Minimalist & content-first** — no flashy SaaS templates, no gradient soup
- **Recruiter-optimized** — key information visible without scrolling
- **Accessible** — semantic HTML, proper heading hierarchy, keyboard navigable
- **Performance-conscious** — lazy-loaded images, minimal bundle size

### Run Locally

```bash
# Clone
git clone https://github.com/1tsadityaraj/Aditya-Portfolio.git

# Install & run
cd Aditya-Portfolio/Frontend
npm install
npm run dev
```

The dev server starts at `http://localhost:5173`.
